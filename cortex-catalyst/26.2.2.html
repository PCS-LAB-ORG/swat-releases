Features Introduced in February 2026 — Release 26.2.2

FeatureDescription
Inline Citation Markers FeatureResponses now display numbered citation markers — [1], [2], [3] — inline with the answer text. Clicking any marker expands the source references panel and highlights the cited document.
Public Source Links FeatureSource citation cards now link directly to the official public documentation page for each referenced document, allowing immediate navigation to the cited content.
Icon Labels FeatureAction buttons in the header now display visible text labels alongside icons, improving discoverability for new users.
Search Quality Improvements EnhancementEnhanced query analysis applies more precise product and category filtering for release notes and version-specific queries, improving result relevance for targeted searches.
Debug Panel FeatureA collapsible per-message debug panel provides request metadata, response statistics, and document relevance scores for each query. Visible in debug-enabled environments only.
